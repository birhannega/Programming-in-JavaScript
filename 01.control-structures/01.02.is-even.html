SRAVAN/SCE0102/is Even

Description

Check the given number Even or Odd.
Create a form that has a text input and a button-'Check Even or Odd'.
When you click the button, you should alert a messege box with the following conditions:
a. If the text field is empty, "Number field is empty"
b. If the text field contains non numeric text, "Only Numbers are allowed"
c. If the text field contains number, "Even" / "Odd"
