SRAVAN/SCE0103/Max of two numbers

Description

Find the max of the given two numbers.
Create a form that has two input fields and a submit button.
When the submit button is pressed,you should display an alert with the following conditions:
a. If both of the text fields are empty, "No numbers given"
b. If any of the fields are not numerics, "Only numberics are allowed"
c. If one field is empty and the other is a number, Display the given number
d. If both the fields are numbers, Display the maximum number
